# Database Module

Drizzle ORM schema definitions and migrations for PostgreSQL 16.

## Structure

- `schema/` — Table definitions (Drizzle ORM)
- `migrations/` — SQL migrations generated by `drizzle-kit`

## Schema Files

| File | Tables |
|------|--------|
| `schema/articles.ts` | `articles` — News article storage |
| `schema/digests.ts` | `digests` — Daily digest entries |
| `schema/sources.ts` | `sources` — RSS feed sources |
| `schema/index.ts` | Re-exports all schemas |

## Conventions

- All timestamps: `timestamp('col', { withTimezone: true })` (TIMESTAMPTZ in UTC)
- UUID primary keys: `uuid('id').primaryKey().defaultRandom()` (server-side `gen_random_uuid()`)
- PostgreSQL-specific types: `text('col').array()`, `jsonb('col')` — no SQLite compatibility
- `source_url` UNIQUE constraint auto-creates index — no redundant indexes
- Migrations generated with `npx drizzle-kit generate`, applied with `npx drizzle-kit push`
